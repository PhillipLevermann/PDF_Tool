# PDF Tool

A lightweight desktop application for merging and splitting PDF files locally.

## Overview
This project was built as part of my portfolio to develop practical Python skills and gain hands-on experience in building desktop applications.

It focuses on structuring applications cleanly and implementing real-world functionality in a simple, usable tool.

---

## Features
- Merge multiple PDF files into a single document  
- Split PDFs using custom split markers  
- Drag & drop file reordering  
- 100% local processing (no uploads required)  

---

## Tech Stack
- Python  
- Tkinter (GUI)  
- pypdf  

---

## What I Learned
- Building desktop GUIs with Tkinter  
- Structuring applications using the MVC pattern  
- Working with files and PDFs programmatically  
- Handling user input and application logic

## Usage
1. Add PDF files to the list  
2. Rearrange files using drag & drop  
3. Insert split markers where needed  
4. Click **Merge** or **Split**  
5. Choose a save location
